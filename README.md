# Flask API

RESTful APIs with Flask. 

## Features

+ Python FastAPI backend.

## How To Use

```console
pip install -r requirements.txt
```

To run the starter:

## console windows
```python main.py```


## For linux server 
``` ./run_app_dev.sh``` OR ```gunicorn --bind 127.0.0.1:8000  --workers=2 wsgi:app &```
